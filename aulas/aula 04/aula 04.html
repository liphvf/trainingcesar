Estudo de caso

Você e seu cliente acordaram que a próxima funcionalidade a ser desenvolvida é a de pesquisa de blog. O formulário de pesquisa deve possuir controles para que o usuário forneça o nome e/ou a descrição do blog.

Roteiro

  1. Crie a página de pesquisa de blog.
  2. Inclua o nome e o menu do BMS na página de pesquisa.
  3. Modifique a aparência no nome do BMS na página de pesquisa.
  4. Modifique a posição do formulário na página de pesquisa.
  5. Modifique a aparência do menu.
  6. Crie o template do BMS.
  7. Aplique o template às páginas de inclusão de blog e de pesquisa.
  8. Associe os controles da página de pesquisa ao managed bean SearchBlog.
  9. Implemente o método searchBlog no managed bean SearchBlog.
  10. Crie o teste searchBlogTest.
  11. Modifique o código para passar no teste searchBlogTest.
  12. Implemente o método selectBlog da classe BlogDao com JPQL.
  13. Modifique a página de pesquisa para exibir os blogs encontrados.
  14. Modifique a página de pesquisa para alternar a cor de fundo das linhas do resultado.
  15. Modifique o formato da data de criação dos blogs.
  16. Crie a classe BMSTestCase refatorando a classe NewBlogTest.
  17. Crie os testes de integração para a funcionalidade de pesquisa de blog.
  18. Configure o Hibernate JPA 2 Metamodel Generator.
  19. Implemente o método selectBlog com a API Criteria.
