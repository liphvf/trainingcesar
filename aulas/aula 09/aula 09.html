Reusando código

Conteúdo

  • Reuso de código
  • Refactoring

Estudo de caso

Você percebeu que o código do CRUD de blogs é muito parecido com o do CRUD de posts. Você prevê que, para desenvolver o CRUD de comentários, desenvolverá, novamente, um código muito parecido. Você deseja reutilizar o código dos CRUDs de forma a tornar mais rápida a implementação do CRUD de comentários, reduzir o esforço com testes e facilitar futuras manutenções.

Roteiro

  1. Crie as classes AbstractEntityDao e AbstractEntitySearchOptions.
  2. Modifique a classe BlogDao para herdar de AbstractEntityDao.
  3. Modifique a classe BlogSearchOptions para herdar de AbstractEntitySearchOptions.
  4. Crie a classe AbstractEntityController.
  5. Modifique a classe BlogController para herdar da classe AbstractEntityController.
  6. Crie a classe AbstractNewEntity.
  7. Modifique a classe NewBlog para herdar da classe AbstractNewEntity.
  8. Modifique a classe NewPost para herdar da classe AbstractNewEntity.
  9. Crie a classe AbstractSearchEntity.
  10. Modifique a classe SearchBlog para herdar da classe AbstractSearchEntity.
  11. Modifique a classe SearchPost para herdar da classe AbstractSearchEntity.

Exercícios

  1. Modifique o OSMS para utilizar o framework.
  2. Adicione o CRUD de produtos ao OSMS.
